# mlb-menu-too

I ran across the old [MLB Menu](https://github.com/markolson/MLB-Menu) and wanted to try and make a modern version.

Add a Developer.xcconfig file with `PRIVATE_DEVELOPMENT_TEAM = [team id]` to configure signing.
